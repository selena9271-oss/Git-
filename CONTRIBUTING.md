# 贡献指南

感谢你愿意改进这个项目。为了让项目保持简单、易打开、易维护，请遵循以下约定。

## 可贡献内容

- 补充新的文件格式说明。
- 修正不准确或表述不清的介绍。
- 改进搜索、筛选和页面交互。
- 优化界面在桌面端和移动端的显示效果。

## 内容规范

新增文件格式时，请至少包含：

- 扩展名
- 类型
- 简要介绍

说明文字应尽量面向普通用户，避免只写技术缩写。

## 开发约定

- 保持项目为纯静态网页。
- 不引入必须安装依赖才能运行的构建流程。
- 修改数据时，请同步考虑 `app.js` 和 `formats.csv` 是否需要一起更新。

## 提交建议

提交信息建议简洁描述变更，例如：

```text
Add quick file extension lookup
Update format descriptions
Improve responsive layout
```
