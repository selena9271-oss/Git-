# 文件格式简要介绍

这是一个本地静态网页项目，用科幻仪表盘风格展示常见文件格式说明。

## 打开方式

推荐先解压整个项目文件夹，然后直接用浏览器打开：

```text
index.html
```

不要在 ZIP 压缩包预览窗口或浏览器临时目录中直接打开 `index.html`，否则样式、脚本和图片可能无法加载。

如果只想发送一个文件给别人，可以使用：

```text
文件格式介绍-单文件版.html
```

## 项目内容

- `index.html`：页面结构。
- `styles.css`：科幻风格界面样式。
- `app.js`：105 条文件格式数据、搜索、分类筛选和详情展示逻辑。
- `文件格式介绍-单文件版.html`：内嵌全部资源的独立版本。
- `formats.csv`：原始表格数据备份，不作为主要展示界面。

## 功能

- 按扩展名、类型、说明内容搜索。
- 输入文件名或选择本地文件，快速识别扩展名对应的格式说明。
- 按文件格式类别筛选。
- 点击格式卡片查看详情。
- 支持“详细”和“简洁”两种浏览密度。
