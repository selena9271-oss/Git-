# 文件格式简要介绍

这是一个本地静态网页项目，用科幻仪表盘风格展示常见文件格式说明。

## 打开方式

直接用浏览器打开：

```text
index.html
```

## 项目内容

- `index.html`：页面结构。
- `styles.css`：科幻风格界面样式。
- `app.js`：105 条文件格式数据、搜索、分类筛选和详情展示逻辑。
- `formats.csv`：原始表格数据备份，不作为主要展示界面。

## 功能

- 按扩展名、类型、说明内容搜索。
- 输入文件名或选择本地文件，快速识别扩展名对应的格式说明。
- 按文件格式类别筛选。
- 点击格式卡片查看详情。
- 支持“详细”和“简洁”两种浏览密度。
