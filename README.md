# 文件格式介绍

一个本地静态网页项目，用交互式界面介绍常见文件格式。项目包含 105 条文件格式说明，支持搜索、分类筛选、快速识别文件扩展名，以及详细/简洁两种浏览方式。

## 使用方式

这是纯前端静态项目，不需要安装依赖。

### 推荐方式：下载后先解压

1. 点击 GitHub 页面绿色 `Code` 按钮。
2. 选择 `Download ZIP`。
3. 下载完成后，先右键压缩包并选择“全部解压”。
4. 进入解压后的文件夹。
5. 直接用浏览器打开：

```text
文件格式简要介绍/index.html
```

不要在压缩包预览窗口或浏览器临时目录里直接打开 `index.html`。那样浏览器可能只临时解出 HTML 文件，导致 CSS、JavaScript 和图片资源无法加载，页面会变成无样式的普通文字。

### 备用方式：单文件版

如果只想复制一个文件给别人，打开：

```text
文件格式简要介绍/文件格式介绍-单文件版.html
```

这个文件已经内嵌样式、脚本和图标，不依赖旁边的 `styles.css`、`app.js` 或 `assets` 目录。

### 在线部署

如果仓库开启 GitHub Pages，可以直接访问站点首页。根目录 `index.html` 会自动跳转到项目页面。

## 功能

- 常见文件格式说明，覆盖文档、图片、音频、视频、压缩归档、数据交换、程序代码、数据库、证书密钥等类别。
- 支持按扩展名、类型或用途搜索。
- 支持输入文件名或选择本地文件，快速识别扩展名对应的格式说明。
- 支持分类筛选和详情查看。
- 支持详细/简洁两种浏览密度。
- 不依赖服务器，不上传用户选择的文件。
